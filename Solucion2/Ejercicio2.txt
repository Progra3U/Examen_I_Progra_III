Para todo ejercicio debe cumplirse:
	Debe estar construido cada ejercicio en soluciones aparte con sus diferentes capas
	Debe utilizar para transición de información entre capaz nada mas objetos o listas de objetos
	Para la conexión y ejecución en base de datos puede utilizar ADO.Net Clasico o Entitiy Framework.

	-Este permitirá registrar las marcas de cada acción y 
	-visualizar por formulario los registros realizados,
